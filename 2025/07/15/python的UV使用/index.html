前言

在使用了一段时间的 miniconda 之后, 最近听说 uv 听说的就试一试, 一下是我的一些记录
uv 是使用 rust 编写的一款 python 环境管理工具, 和 conda 差不多. 不同的是 conda 下载之后自带一个 python, 而 uv 则需要在下载 python 之后使用 pip 下载.

基础操作

下载的方法也是挺简单的 pip install uv ,等待下载完成就可以使用
使用 uv 是以项目为单位进行的,
首先需要初始化一个项目 , 使用 uv init [项目名称] ,uv 会自动创建虚拟环境和基本的配置文件.
其中比较核心的是 pyproject. toml 文件, 里面记录了该项目使用的 python 版本以及依赖
使用 uv run [脚本名称] 来运行脚本
使用 uv add [包名] 来为项目添加需要的 python 包.
使用 uv tree 来显示项目中所有存在的包
使用 uv sync 来更新 pyproject. toml 文件
