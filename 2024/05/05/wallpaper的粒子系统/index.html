wallpaper的粒子系统

粒子系统简介

Wallpaper Engine 具有粒子系统编辑器，可让您创建复杂甚至交互式的粒子系统，并根据自己的喜好对其进行微调。

粒子系统由大量小图像组成，用于构建某些效果，如火、雨或移动物体，如落叶。您可以控制在精灵上渲染的图像、它们的创建方式以及它们的移动方式。您甚至可以使用鼠标与它们进行交互，并且还可以使它们具有音频响应功能。

Wallpaper Engine 允许您在编辑器中现有粒子系统的基础上进行构建，或者从头开始全新的粒子系统。在本文档中，我们将解释粒子系统是如何构建的，以及粒子系统由哪些组件组成，我们将通过几个示例来说明这一点。

粒子系统组件

每个粒子系统都由不同的组件组成，这些组件控制墙纸的某些特征。特别是如果您刚刚开始使用粒子系统，那么在配置粒子系统时打开文档可能会很有意义，以便您知道每个设置的作用。

Component Description
General 定义粒子在屏幕上的渲染方式、使用的纹理以及生成的粒子数量。
Renderers 让我们为粒子定义特殊的渲染器。例如，您可以选择一个特殊的渲染器，该渲染器会根据粒子的速度拉伸粒子。
Emitters 定义创建新粒子的时间和方式。
Initializers 在创建粒子时定义粒子的属性，例如它们的大小、速度和颜色。
Operators 运算符对粒子的属性随时间变化应用更改。使用 Initializer 值作为起点。
Child Particle Systems 子粒子系统是完全独立的粒子系统，它们附加到当前粒子系统。您可以在不同的时间生成它们。可选的
Control Points 控制点是粒子系统或壁纸中可以与之交互的特殊点，包括鼠标光标。可选的

General

粒子系统的“常规”组件定义了粒子在屏幕上的渲染方式以及使用的纹理。
它由三个部分组成：

  • 材质设置
    • 折射
    • 纹理
    • 着色器设置
    • 渲染设置
  • 系统设置
    • 最大计数
    • 开始时间
    • 世界空间
    • 透视渲染
    • Sprite sheet - 动画模式
    • Sprite sheet - 无帧混合
    • Sprite sheet - 序列乘法器
  • 视口设置
    • 显示统计数据
    • 显示轴线
    • 颜色

材质设置

折射

启用后，您可以添加一个额外的法线贴图，Wallpaper Engine 使用该贴图为粒子添加折射效果。

该折射粒子保持在原位，并使用Particle Operator Size Change和Particle Operator Alpha Fade进行动画处理。使用白色图像作为反照率纹理，将材质设置为半透明混合。

纹理

反照率纹理是该粒子系统使用的主要纹理。除此之外,还可以导入sprite sheet以创建动画纹理或生成具有单个纹理的粒子。如果您想了解更多关于如何在粒子系统中使用sprite sheet的信息，请务必阅读我们的粒子系统sprite sheet教程。
法线贴图纹理是可选的纹理贴图.此纹理仅在启用“折射”选项时可见。您可以在此处导入 Wallpaper Engine 将用于折射效果的法线贴图

着色器设置

超亮(Overbright):允许您调整粒子系统的亮度。如果您打算提高亮度，请尝试在壁纸上启用“绽放效果”，尤其是 HDR 绽放可以比增加“过亮”值获得更高质量的结果。
折射量(Refract Amount):可选,此选项仅在启用“折射”选项时可见,用于控制折射效应的强度。

渲染设置

混合(Blending): 根据粒子系统包含的内容类型，选择正确的混合模式非常重要。您可以在“Additive”、“Translucent”和“Normal”混合之间进行选择。

系统设置

最大计数

控制允许生成的最大粒子数。这直接控制粒子分配了多少内存。始终将此值设置为粒子系统运行所需的低值，这可以大大提高性能并减少内存使用。

开始时间

将在创建粒子系统时预先模拟粒子系统一段时间。这有助于避免屏幕看起来是空的，它会启动系统，就好像它已经运行了配置的时间一样。这个数字越高，模拟所需的时间就越长，因为它是基于实时模拟的。

世界空间

如果启用此功能，粒子在创建后将忽略粒子系统的位置、比例和旋转。如果您想将粒子系统附加到另一个对象，这可能很有用,生成的粒子不会与创建它们的对象一起拖动。

透视渲染

透视渲染与 3D 环境相关。对于基于 3D 的壁纸，此选项始终处于打开状态。在 2D 场景中，如果您正在处理 3D 模型，您可以选择将其打开。

Sprite sheet - 动画模式

更改此模式会生成Sprite sheet的随机帧，或者按顺序播放Sprite sheet。

Sprite 工作表 - 无帧混合

禁用帧之间的平滑混合。

Sprite 工作表 - 序列乘法器

这决定了播放精灵表序列的速度。

视口设置

视口设置仅与您在粒子系统编辑器中看到的预览屏幕相关。

显示统计数据

显示右上角的粒子数。用于确定粒子系统“系统设置”中“最大计数”设置的上限。

显示轴

在预览屏幕中渲染水平线和垂直线，以帮助您浏览粒子预览。

颜色

允许您更改预览屏幕的颜色，以在不同的背景上测试粒子。

Renderers

粒子系统至少需要一个渲染器才能运行。有四种不同类型的渲染器可用：

  1. Sprite 渲染器
  2. Sprite Trail 渲染器
  3. Rope 渲染器
  4. Rope Trail 渲染器

每个渲染器都迎合了特定的用例。

Sprite 渲染器

这是在大多数情况下相关的默认渲染器。它会以以配置的方向将粒子渲染到屏幕上

Sprite Trail 渲染器

此渲染器将根据粒子的速度拉伸粒子。
**长度Length:**：长度乘以速度，决定了粒子的理想长度。
最大长度Max Length：粒子在非常高的速度下可以达到的最大长度。

Rope 渲染器

此渲染器在生成的每个粒子之间绘制一条线。
细分：每行中的细分数。数字越大，弯道越平滑，但会牺牲系统性能

Rope Trail 渲染器

此渲染器沿每个粒子的路径绘制一条线。它相对昂贵，但允许您沿着粒子绘制平滑线条，这在复杂的运动场景中可能很有用

Emitters(发射器)

粒子系统的发射器决定了产生新粒子的方式。粒子系统至少需要一个发射器才能运行。

有两种类型的发射器：

  • 盒子随机：在粒子系统原点周围的矩形中生成粒子。
  • 球体随机：在粒子系统原点周围的圆形球体中生成粒子。

虽然不同的发射器类型提供某些独特的设置，但绝大多数发射器设置在所有发射器类型之间共享。

偏移量(offset)

定义粒子生成的每个方向相对于粒子系统位置的偏移量。

Directions 方向

定义粒子在粒子系统原点周围生成的方向。值 表示 1 可以进一步配置的 Distance Max 值
设置为 Y 2 表示发射器的高度是最大距离的两倍。将其设置为 0 意味着发射器没有高度，所有粒子都在一条直线水平线上生成。同样的逻辑也适用于 X 水平值和 Z 深度值（在 3D 壁纸中）。

Sign（球体随机发射器）

此设置允许您强制为给定轴的粒子的生成位置的正值或负值。有效值为：

  • 0 ：默认值，正值和负值均可。
  • 1 ：只能使用正值。
  • -1 ：只能使用负值。

例如：通过设置为 X -1 ，所有 X 值都将为负数，这意味着粒子将仅在粒子系统的左侧生成。请参阅左上角具有不同 X/Y 组合的以下可视化效果：

Distance Min(最小距离)

粒子从粒子系统中心生成的最小距离

Distance Max

粒子从粒子系统中心生成的最大距离。您可以通过调整上述 Directions 选项中的乘数来进一步调整此值。

Control point(控制点)

允许您将发射器位置连接到控制点。有关详细信息，请参阅控制点部分。

Rate

每秒生成的粒子数。粒子的最大数量在粒子系统的“系统”设置中定义。

Duration

发射器的寿命（以秒为单位）。默认值表示 0 发射器永远不会消失。值 为 表示 10 发射器将在 10 秒后停止发射新粒子，但现有粒子将继续存在，直到它们根据其设置消失。

只能通过 SceneScript 重新启动整个粒子系统来重新启动发射器。如果需要，您可以通过 SceneScript 调用粒子系统上的 stop() and play() 函数来满足您的自定义需求。

在壁纸引擎编辑器中，粒子系统将始终无限期可见，您只能在应用和预览壁纸时将其可视化

Instantaneous

创建粒子系统时立即创建的粒子数。根据粒子系统的配置，将生成更多的粒子。请记住粒子的总体最大数量，您可以在粒子系统的系统设置中进行配置。

Limit to one per frame

启用后，发射器每帧仅生成一个粒子。这意味着用户配置的 FPS 限制可能会对粒子数量产生影响。

Speed Max

与移动算子(Operator)结合使用的最大粒子速度

Audio response(音频响应)

设置为 Center 意味着粒子系统将同时对左右音频通道进行音频响应。发射器仅在播放音频时处于活动状态。

Bounds(边界)

边界控制音频响应的开始和停止点。如果音频边界配置为 0 和 1，则音频效果将缓慢淡出。通过使边界更窄，可以实现更突然的效果。例如，如果将音频边界设置为 0.8 和 1，则音频响应能力将仅对介于 0.8 和 1 之间的音量级别生效，这会使音频效果更加突然，并将其限制在音量级别的上限范围内。

Frequency max

效果响应的最高音频频率。这些值从 0 到 15，其中 0 是低音，15 是高频高音。例如，如果您希望发射器仅对正在播放的音乐的节拍做出反应，请将其设置为 1（因为节拍通常由较低的低音频率反射）。

Frequency min

效果响应的最低音频频率。这些值从 0 到 15，其中 0 是低音，15 是高频高音。例如，如果您希望发射器仅对正在播放的音乐的节拍做出反应，请将其设置为 0（因为节拍通常由较低的低音频率反射）。
